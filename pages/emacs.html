Emacs : The Basics

This tutorial has been written as a reference for a session on emacs that I am conducting on 20th January, 2014, as part of my job as a teaching assistant for 15-122
Here I intend to highlight the basic commands for various essential tasks as well as a few select commands that I have found extremely useful, as a programmer who writes code exclusively in emacs.
Any commands mentioned are to be run in a shell on a Mac or Linux environment. In commands, 'C' refers to the Ctrl key and 'M' refers to the Alt or Option key. The command 'C-x' means first press the Ctrl key, then press 'x' while keeping the Ctrl key pressed.

Opening and closing files

To open a file foo.c0 in emacs, simply type the following command in the shell emacs foo.c0
To close a file in emacs, use the command C-x C-c. If you have made any changes to the file you will be prompted to choose one of many options. The only ones that usually concern you, will be 'y' for yes (to save your changes) and 'n' for no (to discard your changes).

Saving files

To save the file, use C-x C-s

Navigating

By Word
Forward — M-f
Backward — M-b
By Line
Next Line — C-n
Previous Line — C-p
Go to beginning — C-a
Go to end — C-e
Go to line num — M-g g num
By Screen
Next Screen — C-v
Previous Screen — M-v

Find and Replace

To find text, use C-s and type in the string you want to search for This highlights all the occurences of the string in the document, starting forward from the current cursor. To move forwards through the matches, use C-s and to move backwards, use C-r.
To replace one string with another, use M-%. When prompted, enter the string to replace and then the string that replaces it. This will replace all occurences ahead of the current cursor

Cut, Copy & Paste

Cut — C-w
Copy — M-w
Paste — C-y

Undo

To undo the previous action, use C-_
